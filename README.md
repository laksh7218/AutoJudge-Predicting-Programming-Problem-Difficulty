# AutoJudge-Predicting-Programming-Problem-Difficulty
AutoJudge helps in predicting the difficulty of problem with the difficulty score as well. The difficulty level is classified into three categories easy, medium and hard and score ranges from 1 to 10 and this prediction is based on the historical data provided to us( in which the rating of easiest problem is 1.1 and of hardest is 9.7)
